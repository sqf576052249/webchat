# webchat

> 基于浏览器端的web实时聊天工具，后台用的java

## 功能列表

1.登录注册

2.添加好友&删除好友

3.发送消息

4.重新连接

5.消息重发

6.上传头像&个人信息

7.附近联系人

8.国际化

## 项目构建

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
